# TextBalance

A **TextBalance** egy Microsoft Word makró, amely segít strukturált, kiegyensúlyozott dokumentumok írásában. Vizuális visszajelzést ad a dokumentum szerkezetéről, a szakaszok hosszáról és arányairól, hogy az írásod minél arányosabb és átláthatóbb legyen.

A tutorial az alábbi linken érhető el
https://textbalance.github.io/tutorial/
